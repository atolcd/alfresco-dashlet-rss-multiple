== Installer le JAR sur Alfresco 4 ==

1. Arrêter Alfresco
   a. Executer la commande alfresco.sh stop (ou alf_stop.bat selon si vous êtes sous Linux ou Windows)


2. Copier le fichier JAR
   a. Copier le fichier "multiple-rss-feeds-share-dashlet.jar" dans le répertoire "/tomcat/shared/lib/" de votre Alfresco (créer le dossier /lib s'il n'existe pas).


4. Lancer votre Alfresco
   a. Executer la commande alfresco.sh start (ou alf_start.bat selon si vous êtes sous Linux ou Windows)
