Pour récupérer les sources de l'extension :
  svn checkout http://labs.atolcd.com/svn/dashlet-rss-multiple/trunk


Un script Ant est disponible à la racine du projet afin de "packager" la dashlet :
 - dans un fichier JAR (à déposer dans /tomcat/shared/lib)
 - ou dans un fichier AMP.


Pour générer le fichier JAR, il faut exécuter la commande suivante, en se positionnant à la racine du projet :
  ant package-dist-jar

Pour générer un AMP :
  ant package-dist-amp
