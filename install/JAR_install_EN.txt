== Installing JAR for Alfresco 3.4 ==

1. Stop Alfresco
   a. Run alfresco.sh stop (or alf_stop.bat)


2. Copy JAR file
   a. Copy "multiple-rss-feeds-share-dashlet.jar" into the "/tomcat/shared/lib/" folder of your Alfresco (create /lib directory if it doesn't exist).


4. Start Alfresco
   a. Run alfresco.sh start (or alf_start.bat)
